## 🚀 Como Usar

### Instalação Local

1. Clone o repositório:
```bash
git clone <repository-url>
cd sitegustavomatos
```

2. Abra `index.html` em um navegador moderno ou use um servidor local:
```bash
# Python 3
python -m http.server 8000

# Node.js (se tiver http-server instalado)
http-server
```

3. Acesse `http://localhost:8000` (ou a porta indicada)

### Adicionar Imagens

1. Extraia as imagens do PDF do portfólio
2. Coloque as imagens no diretório apropriado em `assets/images/`
3. Atualize os caminhos no `index.html` se necessário
